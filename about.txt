работа со сторонними библиотеками и их оптимизация

для примера берется JQuerry
npm i -S jquery

импорт в index.js
import * as $ from 'jquery';

$('pre').html(post.toString());

применим jquery и в файле аналитики

теперь jquery добавляется в каждый из бандлов тех файлов, которые используют эту библиотеку.

необходима оптимизация. добавим в конфиг webpack
optimization: {
  splitChunks: {
    chunks: 'all'
  }
},

теперь общий код библиотек выносится отдельно
