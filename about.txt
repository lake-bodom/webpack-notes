все файлы приложения перенесены в папку src

проинициализировано приложение npm init

устанавливаем вебпак
npm i -D webpack webpack-cli

создаем файл конфигурации вебпак webpack.config.js

минимальная конфигурация состоит из описания входных и выходных данных.
entry: './src/index.js' - входной файл

выходные данные
output: {
  filename: 'bundle.js', - название конечного бандла
  path: path.resolve(__dirname, 'dist') - путь до директории бандла
}

для запуска сборки, прописать в консоли  npx webpack
по умолчанию сборка происходит в режиме production

для изменения режима сборки в конфигурации добавить mode: 'development' или 'production'

в файле Post.js экспортируем класс по дефолту
импортируем его в index.js

пересобираем бандл.
переносим index.html в папку dist.
вместо подключения каждого скрипта по отдельности, подключаем один единственный bundle.js
но теперь проблема со скриптом аналитики. так как в entry прописан только index.js, который не затрагивает аналитику
