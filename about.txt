добавление webpack-dev-server для автоматической перезагрузки страницы при изменении файлов

npm i -D webpack-dev-server

для его работы в конфиге вебпака прописываем порт, на котором будет работать сервер
devServer: {
  port: 4200
},

а так же добавим скрипт для запуска девсервера в package.json
webpack serve
