на предыдущих этапах стили устанавливались в тег style
теперь вынесем их в отдельный файл.

для этого необходимо установить новый плагин
npm i -D mini-css-extract-plugin

импорт в конфиге
const MiniCssExtractPlugin = require('mini-css-extract-plugin');

добавляем в массив плагинов
new MiniCssExtractPlugin({
  filename: '[name].[contenthash].css'
})

файл стилей будет также иметь в названии хеш

теперь, чтобы применить данный плагин правим правила для css в модулях.
заменяем style-loader на лоадер из установленного плагина
было
{
  test: /\.css$/,
  use: ['style-loader', 'css-loader']
},
